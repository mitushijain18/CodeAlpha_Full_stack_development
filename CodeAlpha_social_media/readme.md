# 📊 Project Management Tool

## 📌 Overview

The **Project Management Tool** is a web-based application designed to help teams efficiently plan, organize, track, and manage their projects. It provides a centralized platform for task assignment, progress tracking, collaboration, and deadline management.

---

## 🎯 Objective

The main objectives of this project are:

* Improve team productivity and collaboration
* Track project progress in real time
* Assign and manage tasks efficiently
* Ensure timely completion of projects
* Provide a structured workflow for project execution

---

## ✨ Features

### 👥 User Roles:

* Admin
* Project Manager
* Team Member

### 📌 Core Features:

* 🗂️ Project creation and management
* ✅ Task assignment and tracking
* 📅 Deadline and milestone management
* 📊 Progress dashboard with status updates
* 💬 Team collaboration and comments
* 🔔 Notifications and reminders
* 📈 Reports and analytics

## 🚀 Installation

```bash id="installpm"
git clone https://github.com/your-username/project-management-tool.git

cd project-management-tool
```

### Frontend Setup:

```bash id="frontpm"
cd frontend
npm install
npm start
```

### Backend Setup:

```bash id="backpm"
cd backend
npm install
node server.js
```

---

## ▶️ Running the Project

* Frontend: `http://localhost:3000`
* Backend: `http://localhost:5000`

---

## 🧠 Workflow

1. Admin creates a project
2. Tasks are assigned to team members
3. Members update task progress
4. Manager monitors progress via dashboard
5. Project is completed and reported
