# 🛒 E-Commerce Web Application

## 📌 Overview

This is a **full-stack E-Commerce web application** designed to provide a seamless online shopping experience. It includes product browsing, cart management, secure checkout, and order tracking features. The project demonstrates real-world implementation of scalable web development concepts.

---

## 🎯 Objective

The main objectives of this project are:

* Build a functional online shopping platform
* Provide smooth user experience for customers
* Implement secure authentication and payment flow
* Manage products, orders, and inventory efficiently
* Demonstrate full-stack development skills

---

## ✨ Features

### 👤 User Features:

* User registration and login
* Product browsing and search
* Add to cart and wishlist
* Secure checkout process
* Order history tracking

### 🛍️ Admin Features:

* Add, update, delete products
* Manage orders and users
* Inventory management
* Dashboard analytics

## 🚀 Installation

```bash id="installecom"
# Clone repository
git clone https://github.com/your-username/ecommerce-app.git

# Move to project directory
cd ecommerce-app
```

### Frontend Setup:

```bash id="frontcom"
cd frontend
npm install
npm start
```

### Backend Setup:

```bash id="backcom"
cd backend
npm install
node server.js
```

---

## ▶️ Running the Project

* Frontend runs at: `http://localhost:3000`
* Backend runs at: `http://localhost:5000`

---

## 🧠 System Workflow

1. User registers/logs in
2. Browses products
3. Adds items to cart
4. Proceeds to checkout
5. Payment is processed
6. Order is confirmed and stored in database
